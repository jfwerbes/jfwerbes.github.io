# jfwerbes.github.io

Personal website of Jovan Werbes — Sr. Technical Program Manager focused on AI transformation.

## Stack

- Plain HTML + [Tailwind CSS](https://tailwindcss.com/) (CDN)
- No build step, no dependencies
- Hosted on GitHub Pages

## Structure

Single `index.html` with six sections: Hero, About, What I Do, AI Toolkit, Journey, Contact.

The hero background is an interactive noise flow field — click anywhere to burst particles into the field.

## Built with

Designed and built with [Claude Code](https://claude.ai/code).
